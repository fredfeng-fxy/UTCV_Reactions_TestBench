Sensor Specifications
  Module power: 5.00V
  Measuring temperature: 0-60 ℃
  Response time: ≤ 1min
  pH sensor with BNC connector
  pH2.0 interface ( 3 foot patch )
  Gain adjustment Potentiometer
  Power indicator LED
